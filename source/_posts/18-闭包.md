## 闭包
函数嵌套函数，内部函数可以引用外部函数的参数和变量，参数和变量不会被垃圾回收机制所收回

## 闭包的好处
1. 希望一个变量长期驻扎在内存当中
2. 避免全局变量的污染
3. 私有成员的存在
```
var aaa=(function(){
    var a=1;
    return function(){
        a++;
        alert(a)
    }
})()

aaa();//2
aaa();//3
```
