> Immediately-invoked Function Expression（IIFE，立即调用函数），简单的理解就是定义完成函数之后立即执行。
比较常见的三种写法：
```
// Crockford's preference - parens on the inside
(function() {
  console.log('Welcome to the Internet. Please follow me.');
}());

(function() {
  console.log('Welcome to the Internet. Please follow me.');
})();

!function() {
  console.log('Welcome to the Internet. Please follow me.');
}();
```
