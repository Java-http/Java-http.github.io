## ***号破坏了继承关系** 

```
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Document</title>
	<style>
		*{
			font-size: 50px;
		}
		// 相当于给所有标签加上font-size: 50px;
		div{
			font-size: 12px;
			//  这里p并不会继承div字体大小样式
		}
	</style>
</head>
<body>
	<div>
		<p>六朵金花分类电脑代理费</p>
		fjdljfld fl
		年发电量了回来的了
	</div>
</body>
</html>
```
