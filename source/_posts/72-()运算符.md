## 分组操作符（），括号内只能包含表达式


```
var num = (1,323,5,6)  //num等于最后一个数；

```

```
(function f(){})()
//加上（），函数声明就可以直接调用了，这是因为（）把里面的函数声明转成表达式了。
```
 //再看一个

```
try {
    (var x = 5); // 分组操作符只能包含表达式，不能包含语句（这里的var就是语句）(x=5)这就是正确的。
  } catch(err) {
    // SyntaxError（因为“var x = 5”是一个语句，而不是表达式——对表达式求值必须返回值，但对语句求值则未必返回值）
  }
```
