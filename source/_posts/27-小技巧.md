## 1.获取数据前，页面显示问题

- 获取数据前：可以用loading显示等待获取
- 获取数据后：如果数据为空，提示“内容为空”

```
<div class="yx-center tip" v-if="form&&!form.length">
```

```
data() {
  eturn {
    form: '' // 初始化为 false 或者 空值，初始化值类型应该与数据类型不一致，避免页面进来的时候就显示“内容为空”的提示
  }
},
```
