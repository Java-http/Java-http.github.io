#### 1.
生成第i个[min,max]区间的随机数，并与之前i-1个数比较，如有重复，令i=i-1;重复生成第i个随机数。
```
function my_ran2(n,min,max){
    var arr=[];
    for(i=0;i<n;i++){
        arr[i]=parseInt(Math.random()*(max-min+1)+min);
        for(j=0;j<i;j++){
            if(arr[i]==arr[j]){
                i=i-1;
                break;
            }
        }
    }
 return arr;
}
```
