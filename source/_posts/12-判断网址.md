如果针对一个具体的URL，光用jQuery，不需要插件可以这样：
```
 $.ajax({
  url: 'http://some.url.com',
  type: 'GET',
  complete: function(response) {
   if(response.status == 200) {
    alert('有效');
   } else {
    alert('无效');
   }
  }
 });

```
